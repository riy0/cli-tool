<h1 align="center">go todo</h1>

## description
This is a cli tool. todo app

### command
`
add, a      add a task
clear, c    clear done tasks
delete, r   delete a task
done, d     done tasks
list, l     task on a list
rename, re  rename a task
sort, s     sort tasks
undone, u   undone tasks
`
